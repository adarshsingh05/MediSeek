## Features

- **Medical Report Processing**: Convert unstructured reports into standardized formats using OCR (Tesseract)
- **AI-Powered Analysis**: Structure report data with Google Generative AI (Gemini)
- **PDF Generation**: Create well-formatted, readable PDFs from raw report data
- **Real-time Communication**: Connect patients and doctors via WebSockets
- **Secure Authentication**: Manage user sessions and permissions
- **Responsive UI**: Modern interface with MUI and TailwindCSS components

## Technology Stack

### Frontend

- React 19 with Vite
- TailwindCSS for styling
- Material-UI components
- Framer Motion for animations
- Socket.io for real-time features

### Backend

- Node.js with Express
- MongoDB for data storage
- Supabase for authentication and file storage
- Google Generative AI (Gemini) for report processing
- Tesseract for OCR capabilities

## Getting Started

### Frontend Setup

```bash
cd MediSeek/frontend
npm install
npm run dev
```
